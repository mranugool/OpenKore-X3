INSTALLATION
============
copy the entire package folder to the openkore main folder
put vanguard.pl in your plugin folder and configure sys.txt to load the plugin

NOTE
====
use this plugin at your own risk
when using "vanguard_hddserial_static 1" you shouldn't log the botted account on with the RO client on the same IP, as this could get you banned
