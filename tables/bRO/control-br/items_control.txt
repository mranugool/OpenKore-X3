# Sintaxe:
# (nome do item) (minimo) (auto-armazém) (auto-venda) [por no carrinho] [pegar do carrinho]
# nome do item          : Nome do item (Digite i no console do openkore para saber o nome correto.)
# minimo                        : Quantidade minima desse item que deseja manter no inventário.
# auto-armazém         : Coloque 1 para voltar ao NPC da Kafra e adicionar esse item ao armazém.
# auto-venda            : Coloque 1 para vender esse item em NPC.
# por no carrinho       : Coloque 1 para adicionar esse item automaticamente no carrinho.
# pegar do carrinho     : Coloque 1 para retirar esse item automaticamente do carrinho.
#
# No exemplo abaixo, vai ser vendido todos os jellopys, armazenar todas as facas e colocar
# todas as flores no carrinho:
#       Jellopy 0 0 1
#       Faca 0 1 0
#       Flor 0 0 0 1
#

# deste item em seu inventário:
#       Poção Vermelha 25 0 0 0 1

# neste arquivo serão armazenados ou vendidos.
#
# Se o auto-armazém e auto-venda estiverem como 1, o Kore irá armazenar primeiro (exceto se
# o armazém estiver lotado), e somente depois irá vender os itens.
#
# O Kore jamais irá vender ou armazenar itens equipados. Por exemplo, se voce estiver usando
# uma faca e na lista ter uma linha de auto-venda de todas as facas, o Kore irá vender somente
# as que estiverem no inventário, jamais a faca que estiver usando.
#
# Não esqueça de definir storageAuto, sellAuto e demais configurações relacionadas
